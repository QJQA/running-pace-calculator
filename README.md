# 🏃 跑步配速计算器 (Running Pace Calculator)

一个简单、现代且科学的跑步配速计算器，专为跑者设计。

## 功能特点

- **多模式距离选择**：支持快捷选择（3km, 5km, 10km, 半马, 全马）或自定义输入距离。
- **智能配速计算**：根据目标距离和期望时间，自动计算平均配速。
- **渐进配速策略**：支持“前慢后快”的科学跑法设置，可自定义渐进刻度（如每 2km）和提速幅度（如每刻度快 5s）。
- **动态调整**：生成的配速表中，每一段的配速均可手动微调，并支持一键重新计算总用时。
- **移动端适配**：响应式设计，在手机上也能流畅使用。

## 技术栈

- HTML5
- CSS3 (原生 CSS 变量，Flexbox/Grid 布局)
- JavaScript (ES6+)

## 部署说明

本项目是一个纯静态网页应用，无需后端服务器，可轻松部署到 GitHub Pages, Vercel, Netlify 等平台。

### Vercel 部署

1. 将代码推送至 GitHub。
2. 在 Vercel 中导入该 GitHub 仓库。
3. 点击 Deploy 即可。

## 本地运行

```bash
# 启动本地服务器 (Python 3)
python3 -m http.server 8000

# 或者直接用浏览器打开 index.html
```
